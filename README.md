## Install python dependencies
```
pip install <PACKAGE_NAMES>
```
nltk,
pickle,
re


### Install gems
```
bundle install --without production
```

### Complete Database Migrations
```
rails db:migrate
```

### Populate database
```
rails db:seed
```

### Run server
```
rails server
```

Your app will run on http://localhost:3000
